# Expense Tracker

A simple front-end expense tracker to add, manage, and track daily expenses.

## Overview
This project helps users record their expenses, view totals, and keep track of spending in a clean UI.  
It focuses on DOM manipulation, basic validation, and saving data locally.

## Features
- Add new expenses (name + amount)
- Display a list of expenses
- Calculate total spending
- Delete an expense
- Basic input validation
- Saves data using LocalStorage (so it stays after refresh)

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project files
2. Open index.html in your browser  
(Or use VS Code Live Server)
